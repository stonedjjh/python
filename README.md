## Guía de Configuración Inicial

Para trabajar de manera aislada y evitar conflictos entre dependencias, se recomienda utilizar entornos virtuales en Python.

### Creación del Entorno Virtual

Se ejecuta el siguiente comando en la terminal para crear el entorno virtual:

```bash
python -m venv env
```

### Activación del Entorno Virtual

*   En sistemas Windows (PowerShell):
    ```powershell
    .\env\Scripts\Activate.ps1
    ```
*   En sistemas Windows (CMD):
    ```cmd
    .\env\Scripts\activate.bat
    ```
*   En sistemas Linux o macOS:
    ```bash
    source env/bin/activate
    ```

### Desactivación del Entorno Virtual

Para salir del entorno virtual activo, se ejecuta:

```bash
deactivate
```
